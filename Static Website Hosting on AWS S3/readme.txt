# Static Website Hosting on AWS S3

This project demonstrates how to host a simple static website using Amazon Web Services (AWS) S3. The website is built using basic web technologies and deployed using AWS cloud storage.

## Features
- Static website hosted on AWS S3
- Simple and responsive webpage
- Demonstrates basic cloud deployment
- Public access configuration for website hosting

## Technologies Used
- HTML
- CSS
- JavaScript
- AWS S3 (Amazon Web Services)

## Project Structure
aws-static-website
│
├── index.html
├── styles.css
└── script.js
## Deployment Steps

1. Log in to the AWS Management Console.
2. Navigate to the S3 service.
3. Create a new S3 bucket.
4. Enable static website hosting in bucket properties.
5. Upload the project files (index.html, styles.css, script.js).
6. Configure bucket permissions and enable public access.
7. Access the website using the S3 website endpoint URL.

## Learning Outcomes
- Understanding of AWS S3 cloud storage
- Basic static website deployment
- Managing bucket permissions and public access
- Introduction to cloud-based hosting
